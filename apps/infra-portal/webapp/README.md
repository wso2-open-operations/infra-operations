## Infra Portal Frontend

### Create config.js file in ./public, with following variable

``` javascript
window.config = {
  APP_NAME: "Infra Portal",
  APP_DOMAIN: "infra",
  ASGARDEO_BASE_URL: "",
  ASGARDEO_CLIENT_ID: "",
  AUTH_SIGN_IN_REDIRECT_URL: "",
  AUTH_SIGN_OUT_REDIRECT_URL: "",
  REACT_APP_BACKEND_BASE_URL: "",
};
```
